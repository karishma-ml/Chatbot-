# AI-Powered Chatbot for Instant Support and Solutions
An intelligent chatbot built with Python, Streamlit, and Ollama (Llama 3.2) that provides instant answers, coding assistance, and learning support through a simple and interactive interface. The application includes user authentication, personalized chat history, and local AI processing, eliminating the need for paid cloud APIs.

## Features
 - AI-powered question answering using Llama 3.2
 - Login & Sign Up authentication
 - Personalized chat history
 - Interactive Help section
